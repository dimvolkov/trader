## Алгоритм торговли (по материалам Уроков 1, 2 и 3)

### Фаза 1: Определение тренда (H1)

1. **Открыть график на таймфрейме H1**, включить разделитель периодов (дни)
2. **Найти визуальную точку начала тренда** — место, где произошёл разворот
3. **Определить тип тренда:**
   - **Восходящий** — 2 повышающихся локальных максимума + 2 повышающихся локальных минимума
   - **Нисходящий** — 2 понижающихся локальных максимума + 2 понижающихся локальных минимума
4. **Разметить импульсы и коррекции** — каждый свинг длится 1–3 дня. Не мельчить — игнорировать внутридневные микроструктуры

### Фаза 2: Проверка продолжения / разворота тренда

5. **Сигнал продолжения тренда:**
   - Восходящий: цена пробивает предыдущий локальный максимум
   - Нисходящий: цена пробивает предыдущий локальный минимум
6. **Сигнал разворота тренда:**
   - Коррекция становится **больше** предыдущего импульса
   - Цена пробивает уровень **начала** предыдущего импульса
   - Если это произошло — прекратить торговлю в старом направлении, ждать формирования нового тренда

### Фаза 3: Построение уровней (только H1!)

7. **Отметить 2 ключевых уровня** для ожидания начала импульса:
   - Уровень **конца прошлого импульса** (= начало текущей коррекции)
   - Уровень **начала прошлого импульса**
8. **Отметить целевой уровень** — ближайший уровень слева на графике (конец/начало предыдущих импульсов/коррекций)
9. **Запрещено:**
   - Использовать уровни со старших таймфреймов (D1, W1)
   - Строить уровни по случайным совпадениям вершин/низин без логики

### Фаза 4: Ожидание паттерна «Слом» (M30)

10. **Определить текущую фазу** — импульс или коррекция?
11. **Ждать завершения коррекции** — цена подходит к одному из двух отмеченных уровней H1
12. **НЕ входить от уровня сразу!** Переключиться на таймфрейм **M30** и ждать паттерн «Слом»
13. **Три сигнала паттерна «Слом»:**
    - **Сигнал 1 (Импульс):** Цена пробивает основание прошлого микротренда M30 в сторону нового тренда H1. Важно: не просто пробила, а продолжила движение дальше
    - **Сигнал 2 (Коррекция):** Цена делает откат (коррекцию) обратно к пробитому уровню. Может быть недобой, касание или небольшое пробитие — это нормально
    - **Сигнал 3 (Новый импульс):** Цена начинает новый импульс в сторону тренда H1, пробивая основание первого импульса (Сигнал 1). Третья точка должна быть дальше первой

### Фаза 5: Вход в сделку

14. **Точка входа** — между Сигналом 2 и Сигналом 3 (на коррекции после первого импульса слома)
15. **Направление сделки — строго по тренду H1:**
    - Восходящий тренд → только покупки
    - Нисходящий тренд → только продажи
16. **Стоп-лосс** — за уровень слома (за основание микротренда M30):
    - Логика: если цена пробивает этот уровень, сценарий разворота отменяется
    - Стоп ставится НЕ для минимизации потерь, а там, где сценарий действительно отменится

### Фаза 6: Управление сделкой

17. **Тейк-профит** — на ближайшем уровне H1 слева (торговый диапазон: от уровня входа до целевого уровня)
18. **При пробое целевого уровня** — формируется новый торговый диапазон до следующего уровня, можно открыть дополнительную позицию
19. **Соотношение риск/прибыль:**
    - Минимум **2:1 – 3:1** при закрытии на ближайшем уровне
    - До **5:1** при удержании на весь импульс H1

### Когда «Слом» НЕ работает (пропускаем сделку!)

| # | Условие | Почему |
|---|---------|--------|
| 1 | Слом сформирован **не на уровне H1** (между уровнями) | Нет опоры — разворот не подтверждён структурой |
| 2 | После слома цена **зафлетила** или слом внутри флэта | Флэт на старшем ТФ отменяет младший сигнал |
| 3 | Слом **без закрепления** (цена улетела на новостях/волатильности) | Нет точки входа с адекватным стопом, не входить «в догонку» |

### Ключевые правила (чек-лист перед сделкой)

| # | Проверка |
|---|----------|
| 1 | Тренд H1 определён и размечен? |
| 2 | Текущая фаза — коррекция подошла к уровню H1? |
| 3 | Сделка открывается **в направлении** импульса H1? |
| 4 | Уровни построены с H1 по началам/концам импульсов? |
| 5 | Паттерн «Слом» на M30 сформирован (все 3 сигнала)? |
| 6 | Стоп-лосс стоит за уровнем слома (обоснованный)? |
| 7 | Торговый диапазон (цель) определён? |
| 8 | R:R >= 2:1? |

### Полная последовательность действий (сводка)

```
H1: Тренд → Импульсы/Коррекции → Уровни (начало/конец импульсов)
                                        ↓
                            Цена подходит к уровню H1
                                        ↓
                         M30: Ждём паттерн «Слом» (3 сигнала)
                                        ↓
                    Вход между Сигналом 2 и 3, стоп за слом
                                        ↓
                   Тейк на ближайшем уровне H1 (торговый диапазон)
```

### Чего НЕ делать
- Не торговать без разметки тренда
- Не торговать против импульса
- Не входить от уровня без подтверждения (паттерна «Слом»)
- Не торговать сломы между уровнями H1
- Не торговать сломы внутри флэта
- Не входить «в догонку» без закрепления
- Не ловить развороты
- Не торговать во флэте по трендовой стратегии
- Не использовать уровни с других таймфреймов
- Не ставить стоп «покороче» — только за уровень отмены сценария
